## How to capture

1. Run `./scripts/start_api.sh`
2. Open http://127.0.0.1:8000/admin
3. Use demo accounts from `DEMO.md`
4. Capture at 1440×900 or 1280×800 (admin) and iPhone simulator (mobile)
5. Optional: export a 30–60s screen recording as `docs/demo-walkthrough.mp4` or host on Loom/YouTube
